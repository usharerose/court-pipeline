# court-pipeline

Data pipeline for historical basketball statistics

## Development Environment Setup

### Prerequisites

As a precondition, please [install Poetry](https://python-poetry.org/docs/1.8/#installation) which is a tool for dependency management and packaging for Python.

### Setup

Clone the repository and set up the development environment:

```bash
git clone git@github.com:usharerose/court-pipeline.git
cd court-pipeline
make setup
```
